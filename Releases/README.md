# 📦 Releases — Архив версий

Эта папка хранит все выпущенные версии Free Karaoke для всех платформ.

## Структура

```
Releases/
└── v1.0.0/
    ├── FreeKaraoke-Windows.zip
    ├── FreeKaraoke-x86_64.AppImage
    ├── FreeKaraoke-Android.apk
    └── checksums.sha256
```

Каждая версия хранится в отдельной папке `vX.Y.Z/`.
