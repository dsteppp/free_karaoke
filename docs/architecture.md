# Архитектура Free Karaoke

## Обзор

Free Karaoke — десктопное приложение с ML-пайплайном для создания караоке из аудиофайлов.

## Компоненты

### core/ — Основное приложение

```
core/
├── main.py                  # FastAPI API-сервер (все эндпоинты)
├── launcher.py              # Точка входа: GPU detection + Token prompt + Huey + Uvicorn + PyWebview
├── tasks.py                 # Huey задачи (очередь обработки треков)
├── ai_pipeline.py           # ML-пайплайн: конвертация, сепарация, Genius, метаданные
├── karaoke_aligner.py       # Aligner V8.4: Whisper + Sequence Matching + VAD
├── aligner_orchestra.py     # Elastic Cluster Alignment
├── aligner_acoustics.py     # VAD-фильтрация
├── aligner_utils.py         # Утилиты выравнивания
├── gpu_detect.py            # Каскадное определение GPU (AppImage)
├── token_prompt.py          # Консольный диалог Genius-токена (AppImage)
├── database.py              # SQLAlchemy модель Track
├── editor_backend.py        # API редактора таймингов
├── huey_config.py           # Конфигурация Huey
├── library_io.py            # Экспорт/импорт библиотеки ZIP
├── app_status.py            # Глобальный статус
├── app_logger.py            # Логирование
├── requirements.txt         # Зависимости (генерируется reinstall.sh)
├── reinstall.sh             # Адаптивный установщик (CPU/NVIDIA/AMD)
├── run.sh                   # Запуск приложения
└── static/                  # HTML/CSS/JS UI
```

**Стек:** Python 3.11, FastAPI, Uvicorn, Huey, PyTorch, Whisper, audio-separator, PyWebview

### releases/ — Скрипты сборки

```
releases/
├── build-appimage.sh        # Универсальный AppImage (2 venv + CUDA libs + Qt6)
├── build-windows.ps1        # Windows Portable (подготовка)
├── _build-cache/            # Кэш загрузок (appimagetool, ffmpeg, CUDA installer)
└── README.md                # Описание процесса сборки
```

### shared/ — Общие спецификации

```
shared/
├── formats/
│   ├── karaoke-lyrics-schema.json  # Схема таймингов караоке
│   ├── library-schema.json         # Схема метаданных библиотеки
│   └── zip-structure.md            # Структура ZIP импорта/экспорта
├── test-data/               # Тестовые данные
└── README.md                # Описание форматов
```

## Поток данных

### Обработка трека
```
Аудиофайл → upload → Huey → Конвертация → Сепарация → Genius → Whisper → VAD → Matching → JSON
```

### Воспроизведение
```
MP3 (Vocals + Instrumental) + JSON тайминги → Плеер → Подсветка слов
```

## Форматы данных

Обе платформы читают и пишут один формат:
- `*(Vocals).mp3` + `*(Instrumental).mp3` — аудио
- `*(Karaoke Lyrics).json` — тайминги
- `*_library.json` — метаданные
- `library.zip` — экспорт/импорт

# Free Karaoke Architecture

## Overview

Free Karaoke is a desktop application with an ML pipeline for creating karaoke from audio files.

## Components

### core/ — Main Application

```
core/
├── main.py                  # FastAPI API server (all endpoints)
├── launcher.py              # Entry point: GPU detection + Token prompt + Huey + Uvicorn + PyWebview
├── tasks.py                 # Huey tasks (track processing queue)
├── ai_pipeline.py           # ML pipeline: conversion, separation, Genius, metadata
├── karaoke_aligner.py       # Aligner V8.4: Whisper + Sequence Matching + VAD
├── aligner_orchestra.py     # Elastic Cluster Alignment
├── aligner_acoustics.py     # VAD filtering
├── aligner_utils.py         # Alignment utilities
├── gpu_detect.py            # Cascading GPU detection (AppImage)
├── token_prompt.py          # Genius token console dialog (AppImage)
├── database.py              # SQLAlchemy Track model
├── editor_backend.py        # Timing editor API
├── huey_config.py           # Huey configuration
├── library_io.py            # ZIP library export/import
├── app_status.py            # Global status
├── app_logger.py            # Logging
├── requirements.txt         # Dependencies (generated by reinstall.sh)
├── reinstall.sh             # Adaptive installer (CPU/NVIDIA/AMD)
├── run.sh                   # Launch application
└── static/                  # HTML/CSS/JS UI
```

**Stack:** Python 3.11, FastAPI, Uvicorn, Huey, PyTorch, Whisper, audio-separator, PyWebview

### releases/ — Build Scripts

```
releases/
├── build-appimage.sh        # Universal AppImage (2 venvs + CUDA libs + Qt6)
├── build-windows.ps1        # Windows Portable (TODO)
├── _build-cache/            # Download cache (appimagetool, ffmpeg, CUDA installer)
└── README.md                # Build process description
```

### shared/ — Shared Specifications

```
shared/
├── formats/
│   ├── karaoke-lyrics-schema.json  # Karaoke timings schema
│   ├── library-schema.json         # Library metadata schema
│   └── zip-structure.md            # Import/export ZIP structure
├── test-data/               # Test data
└── README.md                # Format description
```

## Data Flow

### Track Processing
```
Audio file → upload → Huey → Conversion → Separation → Genius → Whisper → VAD → Matching → JSON
```

### Playback
```
MP3 (Vocals + Instrumental) + JSON timings → Player → Word highlighting
```

## Data Formats

Both platforms read and write the same format:
- `*(Vocals).mp3` + `*(Instrumental).mp3` — audio
- `*(Karaoke Lyrics).json` — timings
- `*_library.json` — metadata
- `library.zip` — export/import
